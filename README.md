# SQL-hotel-management-system

This repository contains 200 SQL queries solved using a Hotel Management System database. These queries cover various aspects of hotel operations, including customers, bookings, rooms, staff, and payments.

The purpose of this repository is to practice and demonstrate SQL skills, including:

● Data retrieval (SELECT)
● Filtering and sorting (WHERE, ORDER BY, LIMIT)
● Aggregations (COUNT, SUM, AVG, MAX, MIN)
● Grouping and HAVING
● Subqueries
● Joins (INNER JOIN, LEFT JOIN, SELF JOIN)
● String functions (CONCAT, CONCAT_WS)
● View creation (CREATE VIEW)
● Conditional queries and pattern matching (IN, LIKE)

Key Features :
● Queries for data analysis, report generation, and management insights
● Examples of aggregations, subqueries, and joins
● Demonstrates practical use cases in hotel management
● Good reference for SQL learning and interview preparation
